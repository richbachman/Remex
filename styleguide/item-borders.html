2 Borders

The Border modifier is a four part modifier including the border style, border position, border size, and if the border is rounded. The border type and border size modifier must be used together. The border color is controlled by the $border color variable. That can be adjusted in the _colors.scss file. The code below outlines each border type, and also includes border size and rounded corner examples.
