9 Flexbox Grid

The Flexbox Grid allows you to build the structure of your page. It is a 12 column grid with responsive classes for xs, sm, md, and lg screen sizes. Below is a very basic example of the full grid. There are a lot more options available when using this grid. For the full grid documentation, please visit http://hugeinc.github.io/flexboxgrid-sass/ This version of the grid is a slight variation, i.e. for .col-sm-6 use .col__sm__6.
