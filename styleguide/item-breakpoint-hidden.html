3 Breakpoint Hidden

The Breakpoint modifiers allow you to hide an element on a specific breakpoint. The breakpoints variables are defined in the _include-media.scss file. This file also allows for just about any variation of media query to be written. For full include_media documentation, visit https://include-media.com. Resize the browser to see these modifiers in action.
